## Description
Telegram Poker on TON is a decentralized poker application integrated with Telegram and running on the TON (The Open Network) blockchain. This innovative solution combines the popularity of the Telegram messenger with the security and decentralization of blockchain technology.

## Technologies

Nest.js  
telegraf - [link](https://www.npmjs.com/package/telegraf)  

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
